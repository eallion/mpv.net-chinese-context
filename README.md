# mpv.net 的中文右键菜单

> 匹配版本：[mpv.net v6.0.3.0](https://github.com/mpvnet-player/mpv.net/releases/tag/v6.0.3.0)

安装 [`mpv.net`](https://github.com/mpvnet-player/mpv.net)，替换 `input.conf` 即可。

- 通过 Microsoft Store 安装
- 通过 [GitHub Release](https://github.com/mpvnet-player/mpv.net/releases) 下载安装
- Scoop 安装方式：`scoop install mpv.net`
- Winget 安装方式：`winget install mpv.net`

`input.conf` 一般放在以下路径，选其一：

1. `程序启动目录\portable_config\`

2. `X:\Users%USERNAME%\AppData\Roaming\mpv.net\`

3. Scoop 目录：`X:\Scoop\apps\mpv.net\current\portable_config\`

4. 也可以直接从 `mpv.net` 的右键菜单中打开配置文件夹： `Settings` > `Open Config Folder`
